0 + 1 = SOM


0 + 1 = SOM (meaning 0 + 1 = SOUND) is a project that aims to develop musical and logical skills in young children (aged 6 to 10) using technology.

It consists of a series of online tools and a set of four workshop sessions, in which we work close together with the children, on their classroom.

It was created by Digitópia Casa da Música, in cooperation with Braga Media Arts, and implemented in the Braga Municipality.
